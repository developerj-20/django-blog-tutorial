It's Been A While

July 25, 2017

Hello. It's been a while. I had gotten a subscription to Lynda.com and have been spending all of my free time their website. I've been learning node, webpack, npm scripts, React, more JavaScript, babel, as well as a little PHP and MySQL. I really enjoy working node, webpack, npm, etc. It's different from just using HTML, CSS, and JavaScript and I am almost ready to move in that direction. I've been spending a lot of time learning, but now I need to actually apply everything that I have learned and make something. I'v been a little confused about the route that I should take. Should I focus on React and all that it entails or PHP and MySQL. I know that eventually I am going to be working with databases. So, I had decided to split my time between learning React and learning PHP/MySQL. Between Lynda.com and Pluralsight, I have some pretty good learning resources.

I spend a lot of time learning. I'm a learner. It seems that I'm always either learning something or wanting to learn something. Most of the time, however, I tend to not apply it right them. It may be because I don't know what to do with it at the time or while I was learning it I came across something that I didn't understand and decided to cross-reference. Then I realize I'm now learning something different than what I was initially learning. For example, before I had a Lynda.com subscription, I was trying to learn React. I found out that I should know at least a little about node, so I look into node. Then node is talking about HTTP requests. So I try to learn about HTTP. Now, that's not necessarily a bad thing but I didn't want to get too far away from what I was initially learning, React. So, my goal is to make something. I need to show what I can do.

I did a minor update to the blog, but I am not yet finished. There's more that I want to do to it. I will also be updating my portfolio and adding more relevant projects and things. I have decided to remove Sugar Sugar and just focus on updating this blog as I learn and go further in Web Development. I've been learning more about Git and Sass and will be using them more so that I will get used to using them.

I've got a lot to do. Hopefully, this week I will have something to show for all of the time I've been spending learning. Until next time.
